# 温州大学-大学生网络创新与实践俱乐部简介
大学生网络实践与创新俱乐部（简称“网协”）成立于2009年，是为深化网络工程专业教学改革、促进网络工程应用型人才培养模式的创新，强化大学生网络实践与创新能力而设立的学术科技类社团，曾蝉联学院“三科创新团队奖”。
俱乐部以“温州大学—思科学生实践与创新中心”、“温州大学一思博伦网络测试学生实践与创新中心”、“温州大学中国电信 (温州)学生网络与通信实践与创新中心”等校内实践创新基地以及校外实习基地为依托，
以技术讲坛、开放项目、学生科研、网络大赛、企业见习实习、职业认证培训等多种形式，开阔学生的专业视野，培养学生的专业兴趣，提高学生的实践与创新能力。

## 目录
### 部门职能
#### 科技竞赛部

- 网络工程专业学生的PAT考级培训报销相关工作
- 负责俱乐部会员内部技术性培训等专业建设的开展
- 网络工程科创空间的申请安排等事宜
- 负责工程师、校友讲座的设备调试

#### 联络宣传部

- 活动摄影、文稿撰写、微推制作相关工作
- 负责联系活动相关师生、专家
- 负责日常活动场地的申请
- 负责社团小编号推广

#### 社团内务部

- 负责社团各项活动、会议的组织与开展
- 负责各项活动与讲座策划案与活动记录表的编写
- 社团内部成员的考核与评优评选
- 负责社团资金管理

### 活动开展
#### 网络系列专题讲座

网络实践与创新俱乐部负责许多的优秀校友，以及相关行业的各位大拿的分享讲座，例如我们曾邀请深信服的资深工程师为网络工程专业的学生带来《网络安全及职业新认识》主题的讲座，为现场同学进行答疑解惑，指明了未来方向等；邀请到过去的优秀校友沈健学长介绍自己的各类经历以及感谢，为同学们提供参考；邀请了中国移动资深工程师田上力，给同学们介绍5G组网和优化的基本问题；还邀请了网络工程系的葛一粟老师给同学们介绍人工智能的相关知识。

#### 网协问茶

网络实践与创新俱乐部在5B201以“网协问茶”的形式展开就业问茶活动，邀请了我校网络工程专业优秀学长20网工的张彬、周明轩和17网工的沈晗，与我们共话就业体会。学长们通过他们丰富的学术职场经验，为在场的同学们提供了宝贵的建议。在考研就业问茶活动中，学生们可以了解到考研的最新政策、备考技巧、面试经验等方面的信息，同时也可以了解就业市场的最新动态、求职技巧以及职业规划等方面的内容。通过这类活动，学生们可以更加清晰地认识自己，明确自己的目标和方向，为未来的发展打下坚实的基础。

#### 技能 培训

在“milk-v”开发板的初探活动中，开发板的易用性使得初学者可以轻松上手，学生们可以按照俱乐部提供的实验手册进行体验，通过俱乐部成员的耐心教导，学生们都完成了体验。此次活动让参与的同学能够亲身体验和学习关于开发板的使用和内核编译，加深对操作系统的理解。

#### “破冰”活动

在网协的招新活动结束后，我们组织与新干事们的团建聚餐，以便大家能够更加深入地了解彼此。通过这次聚餐，我们不仅增进了感情，还为未来的合作奠定了坚实的基础。
一年一度的冬至晚会也是我们重要的传统活动。为此，我们精心准备了汤圆和饺子，与干事们共同分享这个充满温暖和团圆的节日。虽然大家身处异乡，但通过这一份特别的聚餐，我们感受到了浓浓的归属感与团圆情。网协大家庭的每一位成员都在这次活动中感受到了亲密与温馨，增强了团队的凝聚力。

#### 毕业晚会

在2024年上半年，俱乐部协助举办了20届网络工程专业同学的毕业晚会，为毕业的同学准备毕业册等精致礼品。在大罗山脚，毕业生们相互道别，俱乐部服务同学们，配合老师们为他们的大学生活画上完美的句号。
